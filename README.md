# MEHEK Fragrances — E-Commerce Website

Front-end e-commerce coursework built with HTML, CSS and vanilla JavaScript DOM manipulation only. No backend, frameworks or external libraries are used.

## Run locally
Open `index.html` in a modern browser, or use a local development server such as VS Code Live Server.

## Features
- Responsive home, shop, product, about, contact, cart and checkout pages
- 21-product JavaScript catalogue
- Search, collection, category and gender filters with Apply Filters
- Add to Bag from product cards and product pages
- Product size buttons for candles and reed diffusers
- Cart quantity, removal, subtotal, shipping and total calculations
- Contact form DOM validation
- LocalStorage cart persistence
- Accessible navigation, focus states, labels and alternative text

## Accessibility evidence
Run Chrome Lighthouse before and after improvements, then save screenshots in the `evidence` folder using the names described there.
